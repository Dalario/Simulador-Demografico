Este projeto implementa um simulador de crescimento populacional baseado na Matriz de Leslie, modelo matemático utilizado para prever o tamanho de uma população distribuída por faixas etárias ao longo do tempo.

A aplicação permite que o usuário defina:
- Taxas de natalidade (aᵢ)
- Taxas de sobrevivência (bᵢ)
- População inicial (X₀)
- Número de períodos de projeção (K)
